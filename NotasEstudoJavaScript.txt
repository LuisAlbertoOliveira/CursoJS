# OPERADORES ARITIMÉTICOS:
	+ : SOMA
	- : SUBTRAÇÃO
	* : MULTIPLICAÇÃO
	/ : DIVISÃO
	% : RESTO DA DIVISÃO
	** : POTÊNCIA

# ORDEM DE PRECEDÊNCIA 
() : Parêntesis
** : Potência
* / %: Multiplicação, divisão e resto da divisão
+ - : soma e subtração

# AUTO ATRIBUIÇÃO 
var n = 3
n = n+4  é o mesmo que n+=4 (simplificando)
n = n-5  <=>  n-=5
n = n*6  <=>  n*=6
n = n/2  <=>  n/=2
n = n**3 <=>  n**=3  (potência)
n = n%5  <=>  n%=5   (resto da divisão)

# INCREMENTO
Ex.: var x = 5
x = x + 1   <=> (o mesmo que) x++
x = x - 1   <=> ( o mesmo que) x--
